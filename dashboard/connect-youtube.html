Connect your YouTube account

TubeStack uses your own free Google Cloud project (one-time setup). You paste one ID into Google, then the same ID into TubeStack. We never store your Google password.

Quick setup (about 10 minutes, one time)
  1. Open Google Cloud Console and create or pick a project.
  2. Enable YouTube Data API v3 (APIs & Services → Library).
  3. Set up the OAuth consent screen (External is fine for personal use). Add yourself as a Test user if the app stays in Testing mode.
