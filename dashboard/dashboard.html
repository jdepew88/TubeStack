Welcome

Install TubeStack and start immediately with local features: save tabs, queues, tags, priority, search, watch-time tracking, and local trends. Google connection is optional and can be added later.

Local vs cloud

TubeStack always keeps your library on this device. Optional cloud pieces are separate — connect only what you want.

  • Local only — Save and close YouTube tabs, queues, tags, priorities, search, watch-time, and local playlists. No Google APIs required.
  • YouTube Data API key — Channel profile scans (handle or URL), recent uploads, and richer public metadata to tune category weights. The key stays on this device.
  • Google OAuth (Client ID + sign-in) — Create real playlists on your YouTube account from saved queues (with the usual Google consent prompt).

You can add an API key or OAuth later in Settings.

Rebuild categories from history

Replaces your library category list with ~25 broad buckets plus up to 15 niches inferred from your YouTube watch history (titles only; same history permission as “Categories from history”). Your library is then re-tagged to the closest new category by keywords.

Categories from watch history

Scans the last 7 days of YouTube watch URLs in Chrome history (titles only on this device). Pick niches to add as green (active) categories for auto-sorting. Run again anytime — this dialog stays off the sidebar.
