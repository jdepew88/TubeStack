TubeStack — yt-dlp native helper (optional)
==========================================

This lets the dashboard button "Download (native helper)" run yt-dlp on your PC
using the embedded Python API (https://github.com/yt-dlp/yt-dlp#embedding-yt-dlp).

1) Install Python 3.10+ and FFmpeg (for merge + audio extract).

2) Install yt-dlp as a module:
     pip install yt-dlp

3) Copy this folder somewhere permanent (e.g. C:\Tools\tubestack-native-host\).

4) Edit com.tubestack.ytdlp.json:
   - "path": ABSOLUTE path to tubestack_ytdlp_host.cmd (or .py if your Chrome
     version accepts it — on Windows, .cmd + python is most reliable).
   - "allowed_origins": set to chrome-extension://YOUR_EXTENSION_ID/
     Find YOUR_EXTENSION_ID on chrome://extensions with Developer mode on.

5) Register the manifest (Windows, current user, Chrome):

   Registry key:
     HKEY_CURRENT_USER\Software\Google\Chrome\NativeMessagingHosts\com.tubestack.ytdlp
   Default (REG_SZ) value:
     Full path to com.tubestack.ytdlp.json

   PowerShell example (run once, adjust paths and extension id):

     $json = "C:\Tools\tubestack-native-host\com.tubestack.ytdlp.json"
     New-Item -Path "HKCU:\Software\Google\Chrome\NativeMessagingHosts\com.tubestack.ytdlp" -Force
     New-ItemProperty -Path "HKCU:\Software\Google\Chrome\NativeMessagingHosts\com.tubestack.ytdlp" -Name "(default)" -Value $json -PropertyType String -Force

6) Reload the TubeStack extension. Downloads go to:
     %USERPROFILE%\Downloads\TubeStack\
